# Login System

## Angular, Node, and MySQL

1. Navigate to backend/config/config.json and change the password to your MySQL password.

```json
{
  "host": "localhost",
  "user": "root",
  "database": "posts",
  "password": "< password >"
}
```
Starting the project : 

2. cd backend
3. \$ npm start
4. cd ../frontend
5. \$ npm start
